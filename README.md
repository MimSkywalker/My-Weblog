# Personal Blog - Django

This repository contains the source code for my personal blog built with Django.  
The project is designed as a learning journey to deepen my understanding of Django, Python, and modern web development practices, while also serving as my personal website and blog.

Key features (planned and implemented):
- Blog posts with categories and tags
- Clean, responsive UI
- Django-based templates and views
- Basic SEO-friendly structure
- Extensible design for future features

Feel free to explore, learn from, or contribute to this project.
